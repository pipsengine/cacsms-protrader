# Documentation

## Project Overview
This project is a Next.js (App Router) application using TypeScript, Tailwind CSS, and modular API structure. It is designed for scalability and maintainability.

## Folder Structure
- `app/` - Main application and API routes
- `components/` - Reusable UI components
- `hooks/` - Custom React hooks
- `lib/` - Utilities and service modules
- `public/` - Static assets
- `.github/` - GitHub Actions workflows and code owners
